## KI als Hauptentwickler: Google setzt auf Maschinen

Sundar Pichai, CEO von Google, hat anlässlich der Cloud Next 2026-Konferenz eine bemerkenswerte Zahl enthüllt: **75 Prozent** aller neuen Zeilen Code bei Google werden von Künstlicher Intelligenz generiert. Im Herbst 2025 waren es noch 50 Prozent.

Das ist kein Pilotprojekt mehr – das ist der neue Standard.

### Branchentrend: Alle wechseln zu KI-Code

Google ist kein Einzelfall:
- **Anthropic** schreibt 70-90% des eigenen Codes mit Claude Code
- **Google** hat eigens ein «Strike Team» gebildet, um die KI-Coding-Fähigkeiten zu verbessern und mit Anthropic gleichzuziehen
- Weitere Tech-Konzerne wie Microsoft und Meta folgen demselben Trend

### Gemini und Siri: Überraschende Allianz

Auf der Cloud Next-Bühne wurde – zu Gelächter im Saal – ein grosses Apple-Logo gezeigt. Hintergrund: Apple integriert Googles Gemini-KI in Siri. Das Upgrade kommt «später dieses Jahr». Für Apple eine ungewohnte Abhängigkeit von einem direkten Konkurrenten.

### Was bedeutet das für die Software-Industrie?

Die Verschiebung ist fundamental: Entwicklerinnen und Entwickler schreiben immer weniger Code selbst – sie reviewen, testen und steuern KI-Output. Die Frage ist nicht mehr «Kann KI Code schreiben?», sondern «Wer haftet, wenn KI-generierter Code Fehler enthält?»

## Einordnung

Für ein Schweizer Softwareunternehmen wie SwissMentor AG ist das ein wichtiges Signal: KI-gestützte Entwicklung wird zum Standard, nicht zur Ausnahme. Wer jetzt nicht in KI-Tooling investiert, verliert an Effizienz gegenüber grossen Playern. Gleichzeitig: Qualitätssicherung und Code-Review werden wichtiger, nicht weniger.
